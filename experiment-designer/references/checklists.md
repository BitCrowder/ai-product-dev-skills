# Experiment Design Checklist

- Hypothesis is falsifiable.
- Experiment type matches risk and learning goal.
- Primary metric is not confused with guardrails.
- Randomization unit is explicit.
- Exposure event is defined.
- Variants differ only in intended ways.
- Sample pollution and novelty effects are considered.
- Stop rules are defined before launch.
- User harm and support impact are considered.
- Decision rules are clear enough to avoid post-hoc interpretation.
